a INTRODUCTION TO HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.

HISTORY OF HTML

The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000's was HTML 4.01, which became an official standard in December 1999. Another version, XHTML, was a rewrite of HTML as an XML language.

ADVANTAGES OF HTML

HTML helps to build structure of a website and is a widely used Markup language. It is easy to learn. Every browser supports HTML Language. HTML is light weighted and fast to load. Storage of big files are allowed because of the application cache feature.
